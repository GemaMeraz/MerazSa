FlexBox

Gema Nataly Meraz Sánchez 19100218

El eje principal está definido por la propiedad flex-direction, y el eje perpendicular a este.

Contenedor: Es el elemento padre que tendrá en su interior cada uno de los ítems flexibles.
Ítem: Cada uno de los hijos flexibles que tendrá el contenedor en su interior.

flex-container

Propiedad que es lo que hace Valores para la Propiedad
display: flex inline-flex establece un contenedor en línea; flex establece un contenedor en bloque. La propiedad display define un contenedor padre flex -- inline-flex
flex-direction Establece la dirección del eje principal y por tanto la dirección de los elementos flex. row -- row-reverse -- column -- column-reverse
flex-wrap Especifica si puede haber un cambio de línea (wrap) o no (nowrap). nowrap -- wrap -- wrap-reverse
flex-flow Es una forma abreviada (shorthand) para las propiedades flex-direction y flex-wrap. flex-direction -- [flex-wap]
align-items Alinea los elementos del contenedor flex en el eje secundario (cross). flex-start -- flex-end -- center -- baseline -- stretch
align-items Alinea los elementos del contenedor flex en el eje secundario (cross). flex-start -- flex-end -- center -- baseline -- stretch
justify-content Alinea los elementos del contenedor flex en el eje principal, evidentemente cuando no ocupan todo el espacio. flex-start -- flex-end -- center -- space-between -- space-around
align-content Alinea los elementos del contenedor flex cuando los elementos ocupan más de una línea y no utilizan todo el espacio disponible en el eje secundario flex-start -- flex-end -- center -- space-between -- space-around

flex-items

Propiedad que es lo que hace Valores para la Propiedad
align-self Reposiciona elementos individuales relativamente al eje transversal auto -- flex-star -- flex-end -- center -- baseline -- stretch
flex-grow Define cuánto puede aumentar un elemento flex. initial -- inherit -- numero
flex-shrink Define cuánto puede disminuir un elemento flex initial -- inherit -- numero
flex-basis Especifica el valor inicial del tamaño principal de un elemento flex initial -- inherit -- auto -- numero
order Se utliza para cambiar el order en el que aparecen los elementos individuales initial -- inherit -- auto -- numero
flex La forma abreviada para flex-grow, flex-shrink y flex-basis none -- flex-grow [flex-shrink] [flex-basis]
